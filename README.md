# HDCinema
Cinema，支持各大视频网站vip视频解析观看。

## 基本操作
选择你要观看的会员视频或电视剧，点击右上方线路按钮跳转到线路解析界面，然后选择解析线路自动回到准备播放的页面。（看进度条是否走完等待解析就行，如果不想等待就操作别的选择，如果网络不好或无网络是无法完成操作。解析成功自动复制链接地址可用于电脑浏览器观看。）
